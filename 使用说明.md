# 晚安广师君
## 写在前面 :smile:
该项目由晚安广师君团队（大二团队）独立开发，项目作为一个校园(GPNU)论坛类型的独立微信小程序云开发项目，包含漂流瓶、各种小工具（未完善）、
论坛发表点赞关注等基本操作 另外可设置管理员实时对不良言论进行处理，欢迎fork star :star: 

## 开发技术 :zzz:
- 微信云开发 
- van组件库
- js基本语法
- ...
## 如何部署？
:one: 使用微信开发者工具选择云开发新项目-直接打开下载的压缩包文件  
:two: 点击工具--构建npm  
:three: 云开发数据库中新建以下几个库 
 - huati
 - plp  
 - sshuo  
 - system   
 - user
